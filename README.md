# Jobs markup demo

A static HTML files for jobs website.

Requirements and Tools used:

-XAMPP with enabled Apache server + MYSQL DB

Application should be run from:

index.php

Functionalities:

1)Guests
- see all posted offers on main page
- create new offers
- register as admins

2)Admins
- all of the above
- use Edits panel to :
    - edit offers
    - delete offers
